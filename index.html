Download BetterBlox

How to Install

  1. Download the BetterBlox ZIP file using the button above.
  2. Extract the ZIP to a folder on your computer.
  3. Open Chrome and go to chrome://extensions/.
  4. Enable Developer Mode (toggle in the top-right).
  5. Click Load unpacked and select the extracted BetterBlox folder.
  6. Go to Roblox and enjoy BetterBlox!
